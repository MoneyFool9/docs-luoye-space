## 文件结构参考

### Phase 1：简单启动

```
src/
├── tokens/
│   ├── index.ts           # 统一导出
│   ├── colors.ts          # Primitive 颜色
│   ├── spacing.ts         # 间距刻度
│   └── semantic.ts        # 语义 Token
├── styles/
│   └── globals.css        # CSS 变量（Light/Dark）
├── hooks/
│   └── useTheme.ts        # 主题切换
└── tailwind.config.ts
```

> 详见 [[设计系统-05-工程化落地方案#阶段一方案：直接 JS/TS 定义]]

```
packages/

│   └── style-dictionary.config.js

```

## 常用代码片段

### CSS 变量定义

```css
:root {
  --color-bg: 255 255 255;
  --color-text: 17 24 39;
  --color-primary: 59 130 246;
}

.dark {
  --color-bg: 15 23 42;
  --color-text: 248 250 252;
  --color-primary: 96 165 250;
}
```

> 完整示例见 [[设计系统-04-多主题与暗黑模式最佳实践#CSS 变量定义]]

### Tailwind 配置

```javascript
module.exports = {
  darkMode: 'class',
  theme: {
    colors: {
      background: 'rgb(var(--color-bg) / <alpha-value>)',
      foreground: 'rgb(var(--color-text) / <alpha-value>)',
      primary: {
        DEFAULT: 'rgb(var(--color-primary) / <alpha-value>)',
        foreground: 'rgb(var(--color-primary-foreground) / <alpha-value>)',
      },
    }
  }
}
```

> 完整配置见 [[设计系统-02-Tailwind-CSS设计哲学与深度定制#完整的 tailwind.config.js 示例]]

### 主题切换 Hook

```typescript
const { theme, setTheme } = useTheme();
// theme: 'light' | 'dark' | 'system'
```

> 完整实现见 [[设计系统-04-多主题与暗黑模式最佳实践#主题切换 Hook]]
