## 一、这套攻略的特点

- **不海选题目**：README 即刷题路线，按顺序刷即可。
- **全链路**：每个专题含「理论基础 → 实战题目 → 总结复盘」。
- **经典高频**：题目多为面试高频与典型考点。
- **多语言**：讲解以 C++ 为主，社区有 Java / Python / Go / JavaScript 等实现。

**建议学习顺序**：先刷「数组 → 链表 → 哈希 → 字符串」，再进阶「二叉树 → 回溯 → 贪心 → 动态规划 → 图论」。

## 三、如何使用本仓库的笔记

1. **按模块顺序学**：从「01-数组与双指针」开始，每个文档里都有**理论基础要点**和**核心题思路**，先理解再做题。
2. **配合原仓库**：题解、多语言代码、图示以 [leetcode-master](https://github.com/youngyangyang04/leetcode-master) 为准，本笔记做**提炼与自查**。
3. **及时复盘**：每个模块末尾有「总结」或「易错点」，刷完题回来扫一眼，形成闭环。
