# 网络请求学习路径

> 目标：从 XHR 入门，到 fetch/axios 实战，建立可复用的请求规范。

## 学习顺序（建议）
1. AJAX / XHR 基础（理解请求-响应、状态码、readyState）
2. fetch 基础（Promise 与错误处理）
3. axios 工程化配置（拦截器、统一错误、超时、取消）
4. 实战排错（Network 面板、CORS、鉴权）

## 路径目录
- 入门：[AJAX 网络请求相关](./ajax网络请求相关)
- 进阶：[fetch / axios 配置与常用写法](./fetch%20axios配置)

## 实战清单
- 统一 baseURL / headers（含 token）
- 统一错误处理（4xx/5xx）
- 超时 / 取消
- 上传进度 / FormData
- 调试：Network 面板 + 响应体

## 快速自测
- 能说出 2xx/4xx/5xx 含义
- 能用 fetch 处理 4xx/5xx 错误
- 能用 axios 拦截器统一返回结构
- 能定位 404 / 401 / 500 问题原因
